Pokemon API

An api for basic pokemon data.

Base URL

https://misqke-pokemon-api.herokuapp.com/api/

Routes

  • search
  • random

Search Route

Get Requests

returns a data object of a single pokemon whos number matches the num query.

query type default desc.
num number 1 number of requested pokemon

Post Requests

returns data object containing keys for "pages" (number) and "pokemon" (list of pokemon objects)

query type default desc.
minNum number 1 lowest pokemon number to include
maxNum number 898 highest pokemon number to include
page number 1 desired page number to return
limit number 12 number of pokemon to return
sortBy string 01 how to sort data:
  • "01" - lowest number to highest
  • "10" - highest number to lowest
  • "az" - alphabetical a to z
  • "za" - alphabetical reverse, z to a
